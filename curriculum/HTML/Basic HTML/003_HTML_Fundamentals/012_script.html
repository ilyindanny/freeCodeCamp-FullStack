Маленький совет на будущее:

Чтобы не запутаться в путях при более сложной структуре, всегда помни:

  • src="something.js" — ищет в той же папке, где index.html
  • src="folder/something.js" — ищет в подпапке
  • src="../something.js" — поднимается на уровень вверх
  • ="/something.js" — ищет в корне сервера, а не проекта (на локалке не работает без сервера!)
